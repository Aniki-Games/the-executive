    Actions and Conditions

    Arithmetic conditions:

    All arithmetic conditions take two parameters: - "op": what operation to perform on the checked stat. Can be "<", "<=", "=", "!=", ">=", ">" - "value": the value to compare the checked stat to

    Global conditions

    { "if": "check-marker",          "name": "marker-id", "op": ">=", "value": 5 }
{ "if": "chance",                "threshold": 1 }
{ "if": "time",                  "duration": "5" | "after": "5" | "before": "5" }
{ "if": "time",                  "after": "1980/3" }
{ "if": "quest-stage-done",      "state": "any/success/failure" }
